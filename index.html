Shape Inheritance Project

This repository contains a Java Object-Oriented Programming project developed using NetBeans.

Project Features

  • Abstract Class (Shape)
  • Inheritance
  • Polymorphism
  • Method Overriding
  • Custom Exceptions
  • Dynamic Binding

Classes Included

  • Shape.java
  • Circle.java
  • Rectangle.java
  • Triangle.java
  • InvalidShapeException.java
  • ShapeDriver.java
